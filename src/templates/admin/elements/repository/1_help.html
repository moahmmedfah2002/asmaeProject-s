• Short Name
    • The Short Name will be used in the if you are using Path mode. Eg. www.example.com/short_name/article/1/
  • Object Name
    • You can tell Janeway what type of objects are being deposited. Some examples are Preprints or Postprints.
    • You should also provide the plural form of the object name.
  • Publisher
    • The publisher name is used across the site and as the Publisher for Crossref deposit and Dublin Core metadata output. It should be the name of your organisation for example: Birkbeck, University of London
