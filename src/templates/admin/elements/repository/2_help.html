• About
    • The contents of About will appear as a CMS style page. If it is left blank this page will not appear.
  • Logo
    • Your logo will appear in the top left hand corner of the front end. We recommend it be landscape and around 80px in height. If you do not add a logo the Repository Name will be displayed in its place.
  • Favicon
    • Upload an .ico file to display a favicon in the browser tab. Alternatively do this via apache/nginx/webserver config to avoid the overhead of loading WSGI for an icon.
  • Hero Background
    • You can upload an image here to overwrite the default hero image on the Preprint display page.
  • Footer
    • This field allows you to customise the footer of your repository.
  • Account Page Text
    • This field allows you to inject text into the top of the Login and Register forms.
  • Custom JS Code
    • This field allows you to inject any custom Javascript code into the footer area of your site.
  • Review helper
    • If you intend to invite reviewers to comment on your {{ request.repository.object_name_plural }} you should add guidance here.
